# Java

## 推荐课程

1. [CS61B: Data Structures and Algorithms](https://sp24.datastructur.es/) 是 UC Berkeley CS61 系列课程之一。因为想学习一下 Java，我看过几节课。但是我注册好 UC Berkeley 的 Gradescope 账号之后，在 GitHub 上拉取代码时，不知道为什么总是无法像官网所说的那样顺利运行，来来回回折腾了四五个小时，最终还是放弃了。这并不是说这门课程不好。相反，很多社区都非常推荐它。它有丰富的 lab 和 project，以及完备的课程资料，能让你比较直接地接触到软件工程的基本面貌和工业级代码。另外，这门课程也不完全像名字看起来那样只讲数据结构与算法，它会先带你学习基础语法，再逐步过渡到数据结构与算法，因此推荐给有志于从事软件开发的同学参考。
