# Python  
## 推荐课程

1. [CS61A: Structure and Interpretation of Computer Programs](https://insideempire.github.io/CS61A-Website-Archive/) UC Berkeley 的优质课程，难度较高，适合于想要转码或者对计算机感兴趣的同学。如果没有时间看网课也可以直接看它的配套教材，参考了 SICP：[Textbook](https://www.composingprograms.com/) ，质量非常高！

## 推荐资源

1. [Python 编程（第 3 版）](https://book.douban.com/subject/36365320/)入门推荐，我当时看完了整本书。
2. [廖雪峰的官方网站](https://liaoxuefeng.com/index.html?src=redirect) 适合想进一步学习 Python 的人，类似一本手册，但比官网简洁。
3. [CS61A 学习笔记](https://www.zhihu.com/column/c_1981023312877483099) 知乎上的汉化教程。
