# 前端开发

> 由于本人不擅长前端，因此只能给出一些好用的资源，无法推荐课程。实际上，我曾经感受过全栈，做出一个网站不难，尤其现在有 AI 辅助，但是我觉得前端里面最简单的就是写代码了。更重要的可能是个人审美和具体要求，我常常苦于无法描述我心里想要的效果而非常难受，最终含泪放弃。——徐嘉宁

## 推荐课程

1. [MIT Web Development Crash Course](https://csdiy.wiki/Web%E5%BC%80%E5%8F%91/mitweb/) 顾名思义，这是一个速成课程。不过我觉得如果你无意于从事前端开发，那在 AI 如此发达的今天，浅尝辄止未尝不好。这门课程恰好符合要求，而且我看了 Slides，pretty good。

## 推荐资源

1. [CodePen](https://codepen.io/trending)
2. [HTML5 UP](https://html5up.net/)
3. [Adobe Color](https://color.adobe.com/zh/) 设计师常用的调色盘。
