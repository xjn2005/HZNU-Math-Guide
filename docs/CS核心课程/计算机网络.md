# 计算机网络

> "Across the Great Wall we can reach every corner in the world"——1987 年 9 月 20 日，中国第一份电子邮件内容。

## 推荐课程

1. [CS 168：Introduction to the Internet: Architecture and Protocols](https://sp25.cs168.io/) UC Berkeley 的互联网协议和架构概论。Textbook 写的非常形象，课程难度适中。一共有 3 个 projects，使用 Python 实现，相对来说比较友好。我最近在做（1/3），做完了我会开源。
2. [中国科学技术大学：计算机网络](http://staff.ustc.edu.cn/~qzheng/teaching.html) 中科大的课程历来以「硬核」著称。这门课程的教材是经典中的经典「计算机网络：自顶向下方法」，当然我肯定暂时是没法上这个课😓等我感觉自己还不错的时候会去看这本教材。但这门课程的友好之处在于是国内高校所授，因此寻找资源比较方便。
