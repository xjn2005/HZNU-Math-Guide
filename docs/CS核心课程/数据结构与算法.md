# 数据结构与算法


## 推荐课程

我并没有上过系统的数据结构与算法课程，主要是通过看书和相关网站学习的。因此如果有优质课程欢迎邮件。

## 刷题平台

> 因为 CS 是强调实践的课程，因此适量的刷题是不可避免的内容。事实上，如果不经历刷题的痛苦，你不可能学会数据结构与算法。

1. [Leetcode](https://leetcode.cn/) 经典刷题平台。
2. [Codeforces](https://codeforces.com/) 主要用于算法竞赛。

## 学习资源

1. [《大话数据结构》](https://book.douban.com/subject/35229404/) 精品图解书籍。
2. [《算法图解》](https://book.douban.com/subject/26979890/) 算法入门的精品书籍。
3. [数据结构与算法图解](https://www.zhihu.com/column/c_1927478351389499904) 徐嘉宁的专栏。
4. [Hello 算法](https://www.hello-algo.com/) 数据结构与算法可视化网站，入门精品选择。
5. [VisuAlgo](https://visualgo.net/en) 算法可视化网站。
6. [Data Structure Visualization](http://www.rmboot.com/) 数据结构可视化网站。
7. [labuladong 的算法小抄](https://github.com/labuladong/fucking-algorithm)
8. [灵茶山艾府（灵神）的题单](https://leetcode.cn/discuss/post/3141566/ru-he-ke-xue-shua-ti-by-endlesscheng-q3yd/)
9. [代码随想录](https://programmercarl.com/) 学习算法的网站。
10. [Gragh editor](https://anacc22.github.io/another_graph_editor/) 图论工具。
