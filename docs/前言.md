# 前言

这是一份带有强烈个人风格和喜好的文档，最初诞生于我个人在竞赛与自学过程中的积累。如今，我决定将其完全开放，希望能为后来者提供一些参考作用。

## 发起初衷

本指南的诞生，源于对现状的观察与对开源精神的致敬。我希望通过这个项目，为杭州师范大学数学学院构建一个更加开放、先进的知识空间。

### 1. 建立学院自有的知识生态

国内外顶尖高校往往拥有深厚的自发编写知识库（Wiki）的传统。虽然本项目最初使用容易上手的国内软件「飞书」，但为了追求更极致的​**自由度**​、**可追溯性**以及​**全球开发者的协作体验**​，我最终选择回归 GitHub，以更纯粹的开源方式托管这份指南。

### 2. 规划自学的优质路径

* ​**拒绝重蹈覆辙**​：过去人犯过的错误，现在人还会犯，我想这是需要极力避免的。
* ​**选择先进内容**​：我们致力于引入更具生命力的先进内容，不至于让国内晦涩难懂的教材和落后的知识来消磨学习的热情和兴趣。

### 3. 拓宽视野

只有见识过真正先进的知识体系，才能够领略学科的内核精神所在。只有学习过各种各样的知识内容，才能够发现自己的热爱所在。通过聚合优质课程、连接活跃社区，我们希望帮助每位同学在交流中不断接近自己的心之所向。

### 4. 致敬并回馈开源社区

因为我的知识很多都是从互联网上得来的，很多我素未谋面的老师教授了我很先进的知识。为了致敬他们无私的开源共享精神，我决意开放文档，让所有人都能免费阅读这篇拙劣的作品，以期能对他人有所帮助。

### 5. 对指南使用者所说的话

不用感谢我（如果你想要感谢的话），你应该感谢那些无私开源的教授和他的课题组们，他们将倾注自己心血的内容无私地开源共享在互联网上，没有他们，我也无法知道这些内容，或许也没有办法来进行分享。他们可以说让无数人受益，相比之下，我做的这件事情是微不足道的。希望将来你在他人遇到问题来请教你的时候，也可以不吝惜自己的帮助。

## 我们的价值观

在本站的构建过程中，我们始终坚持以下原则：

* ​**宁缺毋滥**​：我们不追求完备的资料堆砌，我们只在意内容是否足够先进、是否触及本质。
* ​**价值观契合**​：我们重视「开放」、「包容」、「独立」、「思辨」、「共享」、「严谨」，我们不关注你现在的知识水平和能力，我们更希望你认可我们的理念。如果你曾受惠于此，我们期待你在未来也能成为回馈指南的一员。

---

## 警示

最后，请允许我重复这句告诫：**请爱惜自己的羽翼。**

本站提供的便利旨在辅助学习，而非纵容投机。保持**学术诚信**，维护**自己的尊严**是每一位学生的道德底线。

希望这份指南能为你节省下不必要的探索时间，去追求更广阔的**真理**。
