# 检索技巧

AI 时代已经到来，但我发现仍然有很多同学不知道如何进行高效检索。此前在互联网内容爆炸的背景下，无数优质内容散落在网上。平时遇到的很多问题，前人其实都已经遇到过，也留下了解决方案，只是等待你去发现。因此，我认为有必要专门写一篇文章说明如何检索。

## 搜索引擎

进行检索首先需要搜索引擎。世界上最强大、也最知名的搜索引擎当然是 Google，但在国内网络环境下通常无法直接使用。退而求其次可以使用 Bing。至于百度，我个人不太推荐，主要原因是广告过多。如果有条件，还是优先使用 Google。

## 浏览器

基于我对于搜索引擎的喜好，我推荐 Chrome、Edge 和 Firefox。

## 语言

检索时建议先使用中文，如果检索效果不如人意，使用英文往往能给你惊喜。因为英文的优质内容更多，尤其是数理领域。

## 高级检索

一般检索，无非就是把想知道的问题直接输入搜索框，然后回车。最多只是把问题表述得更清晰一点。

那么，我们作为「专业人士」，平时有很多问题需要检索，怎么做能够更高效呢？

我以 Google 为例：

| 指令 | 作用 | 示例 |
| --- | --- | --- |
| `""`（双引号） | 精确匹配，只搜索包含完全相同短语的结果。 | `"人工智能对教育的影响"` |
| `-`（减号） | 排除特定词，过滤掉包含某个词的结果。 | `美洲豹 -汽车` |
| `site:` | 站内搜索，只在特定网站内搜索内容。 | `site:zhihu.com 咖啡机` |
| `filetype:` | 指定文件格式，搜索 PDF、PPT、Word 等文档。 | `2025 年行业研究报告 filetype:pdf` |
| `intitle:` | 要求关键词必须出现在标题中。 | `intitle:食谱 意面` |
| `after:` / `before:` | 按时间过滤，只看特定日期之后或之前的网页。 | `AI 趋势 after:2025-01-01` |
| `OR` | 逻辑「或」，搜索包含 A 或 B 的内容。 | `考研 OR 保研` |
| `*`（星号） | 通配符，代表一个未知的词。 | `一寸光阴一寸*` |

至于如何找到这些指令：可以参考各个搜索引擎的「检索指南」。

## AI 的使用

随着 AI 在 2025 年之后快速普及，现在很多人习惯于什么都先问 AI。这本身没什么问题，但由于部分网站的隐私限制、内容封闭和检索盲区，AI 并不能完全代替你，你仍然需要具备基础的检索能力。

不过我非常乐于讲一讲如何使用 AI。

首先是选择什么 AI。根据我个人的体验，**ChatGPT** 和 **Gemini** 确实更好用，但在国内使用不太方便。国内我觉得相对好用的是 **豆包**。

其次，是怎么写 AI 的提示词。好的 Prompt 能让你事半功倍，而差的 Prompt 往往只会让你浪费时间。考虑到很多人现在主要使用豆包，这里有一个关于豆包提示词的[仓库](https://github.com/langgptai/awesome-doubao-prompts)，可以参考。

最后我想说的是，AI 的到来确实有利于我们更新学习和生活的方式，但这不意味着我们可以停止思考。恰恰相反，我们需要更深入的思考、更主动的求知和更包容的心态，因为这些能力构成了人与 AI 的根本区别。
