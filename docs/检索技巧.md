# 检索技巧

AI 时代浩浩荡荡的来了，但是我发现竟然有这么多的同学不知道如何进行高效的检索！此前在大数据的浪潮之下，无数的优质内容散落在互联网中。而平时遇到的那些问题，大部分前人都已经遇到过，而且留下了解决方案，只是留待你去发现。因此，我认为很有必要来写一篇文章说明如何检索。

## 搜索引擎

进行检索首先需要的就是搜索引擎。世界上最高效和最知名的是 Google，但很可惜在国内的网络环境之下你无法使用，退而求其次则是 Bing，我不喜欢百度，因为全是广告😓但怎么说呢，如果有条件还是优先 Google。

## 浏览器

基于我对于搜索引擎的喜好，我推荐 Chrome、Edge 和 Firefox。

## 语言

检索时建议先使用中文，如果检索效果不如人意，使用英文往往能给你惊喜。因为英文的优质内容更多，尤其是数理领域。

## 高级检索

一般检索可能就是把想知道什么直接打在搜索框，然后回车。最多就是把文字表达的更清晰。

那么，我们作为「专业人士」，平时有很多问题需要检索，怎么做能够更高效呢？

我以 Google 为例：

| 指令             | 作用                                         | 示例                                         |
| ------------------ | ---------------------------------------------- | ---------------------------------------------- |
| "" (双引号)      | 精确匹配：只搜索包含完全相同短语的结果。     | "人工智能对教育的影响"                       |
| - (减号)         | 排除特定词：排除包含某个词的结果。           | 美洲豹 -汽车 (只搜动物，不搜车)              |
| site:            | 站内搜索：只在特定网站内搜索内容。           | site:zhihu.com 咖啡机                        |
| filetype:        | 指定文件格式：搜索 PDF、PPT、Word 等。       | 2025年行业研究报告 filetype:pdf              |
| intitle:         | 标题包含关键词：要求关键词必须出现在标题中。 | intitle:食谱 意面                            |
| after: / before: | 按时间过滤：只看特定年份之后或之前的网页。   | AI 趋势 after:2025-01-01                     |
| \*\*OR / \`      | \`\*\*                                       | 逻辑“或”：搜索包含 A 或 B 其中之一的内容。 |
| \* (星号)        | 通配符：代表一个未知的词，适合找歌词或名言。 | 一寸光阴一寸\*                               |

至于如何找到这些指令：可以参考各个搜索引擎的「检索指南」。

## AI 的使用

因为自 2025 年以来 AI 的蓬勃发展，现在的人很倾向于什么都问 AI，这没什么问题，不过由于部分网站的隐私规定，AI 也不能完全代替你，你仍需要有基础的检索能力。

不过我非常乐于讲一讲如何使用 AI。

首先就是选择什么 AI。根据我个人的感觉**​ ​Chatgpt**​**​ ​**和**​ Gemini ​**确实是更好用，但可惜国内无法使用。国内我觉得好用的 AI 是​**豆包**​。

其次，怎么用 AI 的提示词。好的 Prompt 能让你事半功倍，而差的……你可能会问候 AI 的家人。考虑到大部分人只能使用豆包，这是一个关于豆包提示词的[ repo ](https://github.com/langgptai/awesome-doubao-prompts)可以参考。

最终，我想和大家说的是。AI 的到来确实有利于我们迭代学习生活的方式，但这不意味着我们可以不用思考。因为我坚信人类的核心在于思想性，因此我们反而需要更深度的思考、更渴望的求知、更包容的心态，因为这些看似情绪化的事情构成了人与 AI 的本质差别。
