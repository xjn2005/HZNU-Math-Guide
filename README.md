> 这份资料是我在闲暇之余所写，原本用于竞赛帮助同学快速入门。现在决定开放给所有人，其中包含了我之前学习所得到的数据分析知识和一些经验之谈。因此这里面的内容必然带有我强烈的个人风格和喜好倾向，所以仅供参考。  
> 编写这份资料的原因有很多：  
> 第一、国内外众多高校都有自发组织的编写知识库（Wiki）的习惯，我本来是打算用 Github 托管一个 Wiki 的网页，考虑到同学不是计算机专业的学生，因此决定采用国内的软件来构建。
> 
> 第二、我们学校没有类似的知识库系统，这会导致很多同学在自学内容时出现一个问题：不知道从何处着手和不知道资料何从寻找。这会直接导致学生的知识广度和基础素养方面的问题，加之现在国内高校传授的内容还是比较落后，在这些落后的知识和晦涩难懂的教材折磨之下，会让人丧失对于学习的兴趣。而且也会有学生不断重蹈覆辙的问题，过去人犯过的错误，现在人还会犯，我想这是需要极力避免的。
> 
> 第三、在广泛的知识面下，才更容易找到自己想要深入研究的领域。而且了解到优质的课程、社区和有相同兴趣的人交流，有助于能力的提升。
> 
> 第四、因为我的知识很多都是从互联网上得来的，很多我素未谋面的老师教授了我很先进的知识。为了致敬他们无私的开源共享精神，我决意开放文档，让所有人都能免费阅读这篇拙劣的作品，以期能对他人有所帮助。
> 
> 第五、不用感谢我（如果你想要感谢的话），你应该感谢那些无私开源的教授和他的课题组们，他们将倾注自己心血的内容无私地开源共享在互联网上，没有他们，我也无法知道这些内容，或许也没有办法来进行分享。他们可以说让无数人受益，相比之下，我做的这件事情是微不足道的。希望将来你在他人遇到问题来请教你的时候，也可以不吝惜自己的帮助。

**上述是这份文档的初衷，但是我现在觉得这份文档应该具有更广泛的内容。而我个人的兴趣尽管很多，但不可能覆盖到方方面面，因此希望这份文档可以有更多人的参与，我也会尽力邀请同学和老师来帮助这份文档的进一步完善。**  
## 🎯  愿景和目标  
希望这份文档能够成为整个数学学院知识库系统发展的开端，能够供历届学生参考，也可以为老师的教学提供便利。更重要的是，希望这份文档能为你带来帮助，节省不必要的探索时间和沉没成本。同时我也呼吁真正有学习能力和热情的同学能为这份文档做出一些贡献，可以是一门**优质课程**或者是你的**学习笔记**亦或者是你的**博客**和书写的**教程**。  
## ⛳️  知识空间简介
我浏览了国内外的社区和论坛，希望这份文档中能够包含数学学院所学的内容以及相关性较高的学科知识。理想的最终形态应该包含数学、统计学、数据科学、计算机科学以及软件工程的内容。
